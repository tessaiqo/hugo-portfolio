## Описание

Разработала полноценный веб-сайт для тату-магазина с возможностью просмотра каталога, фильтрации и добавления товаров в корзину. Основной упор был сделан на серверную часть и взаимодействие с базой данных. Проект включал создание REST API, документированного с помощью Swagger, и систему регистрации/авторизации пользователей.

### Суть работы

- **Проектирование базы данных** — SQL Server
- **Разработка Web API** на C# (ASP.NET Core) для работы с товарами, заказами и пользователям
- **Реализация аутентификации и авторизации** с помощью JWT токенов
- **Документирование API**  с помощью Swagger/OpenAPI


## Технологии:

- **C#**
- **ASP.NET Core** 
- **Entity Framework** 
- **SQL Server** 
- **Swagger** 

## Результат

Работающий интернет-магазин с возможностью регистрации, входа в систему и оформления заказов.
